## 一、诗人概况

&emsp;&emsp;阿斐，原名李辉斐。1997年开始诗歌创作，1999年第一次发表诗歌，2000年在《下半身》发表作品。诗作见于《中国新诗年鉴》等选本及《诗刊》、《诗选刊》、《诗歌月刊》、《作品》、《天涯》、《诗探索》等刊物，著有电子诗集《以垃圾的名义》、《英雄梦》，执行主编《2004-2005中国新诗年鉴》，
